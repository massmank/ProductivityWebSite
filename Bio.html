Tools Used



The first tool I used was HTML. HTML stands for Hyper Text Markup Language. The main use for HTML is the content of the web page. The basic text written in the web page and the images are all done in HTML. The second tool I used was JavaScript. With JavaScript, you can create a lot of functionality in the web page. The code for adding tasks to the to do list was all done in JavaScript. Lastly, I used CSS for the styling of the web page. CSS stand for Cascading Style Sheet. The good thing about using CSS for styling is that if you want to restyle the web page you can just recode the CSS instead of the entire web page. The CSS contains code for everything from the web page background color to the hover highlight for the thumbnails in the photo gallery.
